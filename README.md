# 🌟 Kashish Dureja | Frontend Web Developer Portfolio

A high-performance, professional-grade portfolio built with **React** and **Vite**, featuring elite visual enhancements, interactive components, and advanced project filtering.

![Portfolio Preview](https://via.placeholder.com/1200x630.png?text=Kashish+Dureja+Portfolio+Preview)

## 🚀 Key Features

- **Elite UI/UX**: Features a breathing blue neon profile glow, glassmorphic elements, and smooth micro-interactions like card-lifting effects.
- **Advanced Project Filtering**: A professional multi-select skill tagging system (React, SVG, HTML, etc.) that allows users to instantly explore projects by specific expertise.
- **Smart Tech Tagging**: Automatic inclusion of core web fundamentals (HTML/CSS/JS) for React-based projects to accurately represent developer skills.
- **Interactive Contact**: Functional greeting form with built-in loading spinners and successful submission notifications with a checkmark feedback.
- **Social Presence**: Deeply integrated LinkedIn profile with official icons and high-quality OpenGraph/Twitter meta tags for premium social media link previews.
- **Performance Optimized**: Built with the latest React 19 and Vite 8 architectures for lightning-fast build and load times (Built in < 1.5s).

## 💻 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/) (v9.0.0 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Kashishportfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Kashishportfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the local development server:
```bash
npm run dev
```
The server will start at `http://localhost:5173`.

### Production Build

Generate a production-ready bundle:
```bash
npm run build
```
The optimized files will be available in the `dist/` directory.

## 🔗 Deployment (Netlify)

This project is optimized for **Netlify**. Follow these steps for deployment:
1. Connect your GitHub repository to Netlify.
2. Set the **Build Command** to: `npm run build`
3. Set the **Publish Directory** to: `dist`
4. Click **Deploy Site**.
